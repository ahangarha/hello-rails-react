 # Hello Rails + React

This is an educational project to practice building  a monolithic Rails+React app.

## Built with

- Ruby on Rails
- React / Redux
- PostgreSQL

### Prerequisites

- Ruby 3+
- Rails 7
- Postgresql
- React

## Setup

- Clone the Repository
- Run `bundle install` and `yarn install`

### Usage

- Run `rails server` to start the server and start the application.
